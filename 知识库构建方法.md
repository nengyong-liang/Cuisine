# 就业知识库构建方法

本文档记录了将 WPS 云盘就业资料批量转换为本地 Markdown/CSV 知识库的完整流程，供学习和后续维护参考。

---

## 一、背景与目标

### 问题

- 就业资料存储在 WPS 云盘中，格式杂乱（.docx / .doc / .xlsx / .xls / .pdf / .pof.wpsonline）
- 不方便维护、搜索和版本管理
- 无法用 AI 工具高效检索和补充内容

### 目标

将所有文件按原始目录结构 1:1 转换为 Markdown 和 CSV 格式，存入 Git 项目，实现：
- 纯文本可搜索
- AI 可读取和补充
- 版本可追踪

---

## 二、技术方案

### 2.1 格式转换策略

| 原始格式 | 目标格式 | 转换工具 | 说明 |
|---------|---------|---------|------|
| `.docx` | `.md` | pandoc（首选）+ python-docx（备选） | pandoc 转换质量高，自动提取嵌入图片 |
| `.doc` | `.md` | Word COM + pandoc | 旧格式，先通过 COM 接口另存为 .docx，再用 pandoc 转 |
| `.xlsx` | `.csv` | pandas + openpyxl | 多 sheet 按 `文件名_Sheet名.csv` 拆分 |
| `.xls` | `.csv` | pandas + xlrd | 旧版 Excel，xlrd 引擎读取 |
| `.pdf` | `.md` | pdfplumber | 纯文本提取，复杂排版可能丢失 |
| `.jpg/.jpeg` | 原样复制 | shutil | 图片文件直接复制 |
| `.pof.wpsonline` | `.md` 占位符 | 手动生成 | WPS 云文档本地无内容，创建占位符提示手动补充 |
| `~$xxx.docx` | 跳过 | - | WPS 临时锁文件，无需处理 |

### 2.2 工具链

| 工具 | 版本 | 用途 |
|------|------|------|
| Python | 3.13.9 | 主脚本语言 |
| pandoc | 3.8 | Word → Markdown 转换（首选） |
| python-docx | 1.2.0 | Word → Markdown 转换（备选，pandoc 失败时降级） |
| openpyxl | 3.1.5 | 读取 .xlsx |
| xlrd | 2.0.2 | 读取 .xls |
| pandas | 2.3.3 | Excel → CSV |
| pdfplumber | 0.11.9 | PDF → Markdown |
| pywin32 | 311 | Word COM 接口（.doc 转 .docx） |

### 2.3 关键技术决策

#### 为什么选 pandoc 而不是 python-docx？

pandoc 的优势：
1. **转换质量高**：正确处理标题层级、列表、表格、加粗/斜体等格式
2. **自动提取嵌入图片**：通过 `--extract-media` 参数，将 Word 内嵌图片提取到 `media/` 目录
3. **处理边界情况**：对复杂排版、嵌套表格等处理更稳健

python-docx 作为备选：
- 当 pandoc 转换失败或输出为空时降级使用
- 只能提取段落文本和简单表格，无法处理嵌入图片

#### .doc 文件为什么用 Word COM？

`.doc` 是旧版二进制格式，pandoc 和 python-docx 都无法直接读取。解决方案：
1. 通过 Windows COM 接口启动 Word 进程
2. 打开 .doc 文件，另存为 .docx（FileFormat=16）
3. 关闭 Word 进程
4. 用 pandoc 将临时 .docx 转为 .md

前提条件：系统安装了 Microsoft Word 或 WPS Office，且 pywin32 可用。

#### Excel 多 sheet 如何处理？

一个 .xlsx 文件可能包含多个 sheet（如国考岗位表按机构类型分 sheet）。处理方式：
- 单 sheet：`文件名.csv`
- 多 sheet：`文件名_Sheet名.csv`（sheet 名中的特殊字符替换为 `_`）

编码选择 `utf-8-sig`（带 BOM），确保 Excel 打开中文不乱码。

## 三、完整转换脚本

```python
"""
批量转换脚本：将 WPS 云盘就业资料目录下的文件转换为 Markdown/CSV 格式
"""

import os
import shutil
import subprocess
import tempfile
import time
import pandas as pd
import pdfplumber

SOURCE_ROOT = r"源目录路径"
TARGET_ROOT = r"目标目录路径"

stats = {"docx": 0, "doc": 0, "xlsx": 0, "xls": 0,
         "pdf": 0, "image": 0, "wpsonline": 0,
         "skipped": 0, "failed": 0, "total": 0}

# ── .docx → .md ──
def convert_docx_to_md(src_path, dst_path):
    # 首选 pandoc
    try:
        result = subprocess.run(
            ["pandoc", "-f", "docx", "-t", "markdown", "--wrap=none",
             "--extract-media", os.path.dirname(dst_path),
             str(src_path)],
            capture_output=True, text=True, encoding="utf-8", timeout=60)
        if result.returncode == 0 and result.stdout.strip():
            with open(dst_path, "w", encoding="utf-8") as f:
                f.write(result.stdout)
            return True
    except Exception:
        pass

    # 备选 python-docx
    try:
        from docx import Document
        doc = Document(src_path)
        lines = []
        for para in doc.paragraphs:
            text = para.text.strip()
            if not text:
                lines.append("")
                continue
            style = (para.style.name or "").lower()
            if "heading 1" in style: lines.append(f"# {text}")
            elif "heading 2" in style: lines.append(f"## {text}")
            elif "heading 3" in style: lines.append(f"### {text}")
            else: lines.append(text)
        for table in doc.tables:
            lines.append("")
            for i, row in enumerate(table.rows):
                cells = [cell.text.replace("\n", " ") for cell in row.cells]
                lines.append("| " + " | ".join(cells) + " |")
                if i == 0:
                    lines.append("| " + " | ".join(["---"] * len(cells)) + " |")
        with open(dst_path, "w", encoding="utf-8") as f:
            f.write("\n".join(lines))
        return True
    except Exception:
        return False

# ── .doc → .md（Word COM → .docx → pandoc）──
def convert_doc_to_md(src_path, dst_path):
    import win32com.client
    import pythoncom
    pythoncom.CoInitialize()
    word = None
    doc = None
    temp_docx = None
    try:
        word = win32com.client.DispatchEx("Word.Application")
        word.Visible = False
        word.DisplayAlerts = False
        doc = word.Documents.Open(os.path.abspath(src_path))
        temp_docx = os.path.join(tempfile.gettempdir(), f"convert_{int(time.time())}.docx")
        doc.SaveAs2(temp_docx, FileFormat=16)  # wdFormatXMLDocument
        doc.Close(False)
        doc = None
        word.Quit()
        word = None
        return convert_docx_to_md(temp_docx, dst_path)
    except Exception:
        return False
    finally:
        try:
            if doc: doc.Close(False)
        except: pass
        try:
            if word: word.Quit()
        except: pass
        pythoncom.CoUninitialize()
        if temp_docx and os.path.exists(temp_docx):
            try: os.remove(temp_docx)
            except: pass

# ── .xlsx / .xls → .csv ──
def convert_excel_to_csv(src_path, dst_dir, base_name, engine):
    try:
        xls = pd.ExcelFile(src_path, engine=engine)
        for sheet_name in xls.sheet_names:
            df = pd.read_excel(xls, sheet_name=sheet_name)
            suffix = f"_{sheet_name}" if len(xls.sheet_names) > 1 else ""
            safe = suffix.replace("/", "_").replace("\\", "_").replace(":", "_")
            csv_path = os.path.join(dst_dir, f"{base_name}{safe}.csv")
            df.to_csv(csv_path, index=False, encoding="utf-8-sig")
        return True
    except Exception:
        return False

# ── .pdf → .md ──
def convert_pdf_to_md(src_path, dst_path):
    try:
        parts = []
        with pdfplumber.open(src_path) as pdf:
            for i, page in enumerate(pdf.pages):
                text = page.extract_text()
                if text: parts.append(text)
                if i < len(pdf.pages) - 1: parts.append("\n---\n")
        with open(dst_path, "w", encoding="utf-8") as f:
            f.write("\n".join(parts))
        return True
    except Exception:
        return False

# ── 主流程 ──
def main():
    for root, dirs, files in os.walk(SOURCE_ROOT):
        dirs.sort()
        for filename in sorted(files):
            src_path = os.path.join(root, filename)
            rel_path = os.path.relpath(src_path, SOURCE_ROOT)
            rel_dir = os.path.dirname(rel_path)
            dst_dir = os.path.join(TARGET_ROOT, rel_dir) if rel_dir else TARGET_ROOT
            os.makedirs(dst_dir, exist_ok=True)

            if filename.startswith("~$"):
                continue  # 跳过临时文件

            ext = os.path.splitext(filename)[1].lower()
            base_name = os.path.splitext(filename)[0]

            if ext == ".docx":
                convert_docx_to_md(src_path, os.path.join(dst_dir, base_name + ".md"))
            elif ext == ".doc":
                convert_doc_to_md(src_path, os.path.join(dst_dir, base_name + ".md"))
            elif ext == ".xlsx":
                convert_excel_to_csv(src_path, dst_dir, base_name, "openpyxl")
            elif ext == ".xls":
                convert_excel_to_csv(src_path, dst_dir, base_name, "xlrd")
            elif ext == ".pdf":
                convert_pdf_to_md(src_path, os.path.join(dst_dir, base_name + ".md"))
            elif ext in (".jpg", ".jpeg", ".png"):
                shutil.copy2(src_path, os.path.join(dst_dir, filename))
            elif filename.endswith(".pof.wpsonline"):
                # 创建占位符
                placeholder = base_name[:-4] if base_name.endswith(".pof") else base_name
                with open(os.path.join(dst_dir, placeholder + ".md"), "w", encoding="utf-8") as f:
                    f.write(f"# {placeholder}\n\n> WPS 云文档占位符，请手动补充内容。\n")

    print("转换完成！")

if __name__ == "__main__":
    main()
```

## 五、踩坑记录

### 5.1 .doc 文件转换

**问题**：`.doc` 是旧版二进制格式，pandoc 和 python-docx 都无法直接读取。

**解决**：通过 Windows COM 接口（pywin32）调用 Word 应用程序，先另存为 `.docx`，再转 `.md`。

**注意**：
- 必须系统安装了 Word 或 WPS
- COM 接口需要初始化：`pythoncom.CoInitialize()`
- 用完必须释放：`pythoncom.CoUninitialize()`
- 使用 `DispatchEx` 而非 `Dispatch`，避免复用已有 Word 进程
- 设置 `word.DisplayAlerts = False`，避免弹窗阻塞

### 5.2 Excel 多 sheet 处理

**问题**：部分 .xlsx 文件包含多个 sheet（如国考岗位表按机构类型分了 4 个 sheet）。

**解决**：按 `文件名_Sheet名.csv` 拆分。单 sheet 文件不加后缀。

**注意**：sheet 名可能包含特殊字符（`/`、`\`、`:`、`?`、`*`），需替换为 `_`。

### 5.3 WPS 云文档

**问题**：`.pof.wpsonline` 文件本地只存储一个链接引用，不包含实际内容。

**解决**：创建 Markdown 占位符文件，标注原始路径，提示手动补充。

**涉及文件**：
- `工作.md`
- `软件网站_金标尺_公考雷达.md`
- `职级、政府体系、政府结构、分级、组织架构.md`
- `重庆长安汽车.md`
- `国有企业_分类_名单.md`

### 5.4 编码问题

**问题**：CSV 文件在 Excel 中打开中文乱码。

**解决**：使用 `utf-8-sig` 编码（带 BOM 头），Excel 可正确识别。

### 5.5 pandoc 嵌入图片

**问题**：Word 文档中嵌入的图片需要提取出来。

**解决**：pandoc 的 `--extract-media` 参数会自动将嵌入图片提取到指定目录的 `media/` 子文件夹中，并在 Markdown 中生成相对路径引用。

### 5.6 空文件处理

**问题**：部分原始文件本身就是空的（如 `2026-实习.xlsx`、`(6)中国兵器工业集团有限公司.docx`）。

**解决**：转换后也是空文件/空 CSV，这是正确行为，不需要特殊处理。

## 六、后续维护

### 添加新文件

1. 判断内容类型，确定目标目录（参考 README.md 中的"AI 添加内容指南"）
2. 按格式规范转换（Word→md, Excel→csv, PDF→md）
3. 放入对应目录
4. 更新 `docs/目录结构.md`

### 更新已有文件

直接编辑对应的 `.md` 或 `.csv` 文件即可。

### 补充 WPS 云文档

打开 WPS 云端，将内容复制到对应的占位符 `.md` 文件中，删除占位符提示。
