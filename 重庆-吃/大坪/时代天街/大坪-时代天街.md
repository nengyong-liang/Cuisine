# 重庆大坪/时代天街/石油路全美食店铺综合统计表-1

通过对您提供的三篇小红书笔记及数百条评论区的深度阅读与数据挖掘，我为您整理了重庆大坪、石油路及龙湖时代天街周边的美食全景图。

### 📊 总体分析概述
1. **社区老店 vs 商场网红**：大坪美食呈现明显的“两极分化”。隐藏在居民楼、菜市场、地铁站口的**苍蝇馆子和夜市摊摊**（如梯坎老火锅、大坪玉肥肠、慕香串串）口碑极佳，是本地人的“食堂”；而商场内的部分**网红“漂亮饭”**（如盐屿、俄士厨房）及过度营销的店铺则成了避雷重灾区。
2. **火锅串串极度内卷**：该区域火锅串串多达数十家，**“响火锅”** 和 **“绮遇米线”** 是提及率最高但争议也最大的店；而 **“郭记明”**、**“四间”**、**“贰妹”** 则凭借性价比和味道成为高赞首选。
3. **性价比天堂**：石油路周边保留了大量平价小吃，如1元串串、7元米线、10元卤肉卷、20元豆花饭，非常适合学生党和打工人。

### 🏆 分类红黑榜总结

#### 🟢 **本地人私藏红榜（闭眼冲）**
* **火锅/串串**：郭记明火锅、梯坎老火锅、慕香红油串串、四间火锅、贰妹贵州地摊火锅。
* **苍蝇馆子/江湖菜**：大坪玉肥肠、阿辉鸡杂、福满轩、悦来香酒家。
* **街头小吃**：重医门口卤肉卷、重医报亭烧烤、乐山油炸串串、天贵热油豆花饭。

#### 🔴 **严重踩雷黑榜（快跑）**
* **卫生/服务极差**：山河烧肉屋（吃出虫子）、隆江猪脚饭（吃出蟑螂）、极品轩北渡鱼庄（耍称、踩人脚）、李讨厌米线（态度恶劣）。
* **难吃/预制菜**：李火辣（发苦）、俄士厨房（拉肚子）、盐屿（预制菜冷掉）、霞姐麻辣烫（辣椒精）、椿哩哩（夹生）。
* **隐形消费/刺客**：百盛怪味鱼（收杀鱼费、极贵）、每味每客（重复算锅底）。

#### ⚖️ **争议/看个人口味榜（酌情尝试）**
* **响火锅 / 陈眼镜火锅**：名气最大，外地游客打卡多，但本地老饕普遍反映“现在变难吃了/锅底苦”，属于“不吃遗憾，吃了也遗憾”的类型。
* **绮遇米线 / 摊摊麻辣串**：味道确实有特色，但摊摊麻辣串存在卫生隐患（多人反馈拉肚子），绮遇米线门口串串阿姨存在“看人下菜碟”坐地起价的现象。
* **平平898**：味道确实是“家常菜之神”，但被网红带火后排队极其恐怖，就餐体验大打折扣。

### 💡 实用觅食避坑指南
1. **警惕“漂亮饭”**：时代天街商场内装修精美、主打拍照的西餐/南洋菜（如盐屿、相思蜜哒），多为预制菜或性价比极低，想吃地道口味请走向**时代汇背街**或**石油路居民楼**。
2. **看差评重灾区**：点餐前务必翻看近期差评，如果集中提到“卫生问题”“食材不新鲜”“服务摆烂”，直接放弃。
3. **保留维权证据**：大坪部分老店（如极品轩、隆江猪脚饭）存在欺客现象，遇到异物或变质，务必保留照片和付款记录，依法主张赔偿。
4. **夜市时间差**：石油路的夜市文化浓厚，但很多神级摊摊（如重医烧烤、摊摊麻辣串）晚上8点后才是最佳觅食时间，去早了可能没出摊或菜品不全。
