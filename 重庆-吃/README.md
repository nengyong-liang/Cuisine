## 🔧 技术细节

### 转换工具链

- **pandoc 3.8**: Word → Markdown（首选）
- **Python 3.13.9**: 主脚本语言
- **pandas + openpyxl**: Excel → CSV
- **pdfplumber**: PDF → Markdown（备用）

### 文件格式说明

#### Markdown 文件 (.md)

每个 `.md` 文件顶部包含 YAML frontmatter：

```yaml
---
source_file: "原始文件路径.docx"
converted_date: "2026-06-12"
---
```

**优势**：
- ✅ 可追溯原始文件来源
- ✅ 记录转换时间
- ✅ 便于后续增量更新

#### CSV 文件 (.csv)

- 编码：`utf-8-sig`（带 BOM，Excel 打开不乱码）
- 多 sheet 处理：`文件名_Sheet名.csv`
- 特殊字符替换：`/` `\` `:` `*` `?` → `_`

#### 图片文件

- Word 内嵌图片：提取到 `media/` 子目录
- 独立图片：原样复制到对应目录
- 保持原始文件名

#### 👤 人类用户

1. **Word 文档** → 转换为 `.md`
   ```bash
   pandoc -f docx -t markdown --wrap=none --extract-media media/ 新文件.docx > 新文件.md
   ```

2. **Excel 表格** → 转换为 `.csv`
   ```python
   import pandas as pd
   df = pd.read_excel("新文件.xlsx")
   df.to_csv("新文件.csv", index=False, encoding="utf-8-sig")
   ```

3. **直接编辑** `.md` 或 `.csv` 文件

4. **提交 Git**
   ```bash
   git add 重庆-吃/
   git commit -m "添加新的美食记录"
   ```
