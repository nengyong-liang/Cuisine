# 快速开始 - 重庆美食知识库

## 🎯 这是什么？

这是一个本地化的重庆美食记录知识库，所有数据从 WPS 云盘迁移而来，格式统一为 Markdown 和 CSV，便于：
- 🔍 **全文搜索**：快速查找餐厅、菜品、区域
- 🤖 **AI 辅助**：让 AI 帮你分析、推荐、总结
- 📊 **版本管理**：Git 追踪每次修改
- 💻 **离线可用**：不依赖 WPS 云端

## 🔍 如何使用

### 1. 浏览目录

直接打开对应区域的文件夹，查看 `.md` 文件。

### 2. 搜索关键词

**Windows PowerShell**:
```powershell
# 搜索包含"火锅"的文件
Get-ChildItem -Recurse -Filter "*.md" | Select-String "火锅"

# 搜索特定餐厅
Get-ChildItem -Recurse -Filter "*.csv" | Select-String "妈也食堂"
```

**Git Bash / WSL**:
```bash
# 搜索关键词
grep -r "火锅" 重庆-吃/

# 列出所有火锅店
grep -r "火锅" 重庆-吃/*.csv
```

### 3. 用 AI 查询

复制相关文件内容，问 AI：

```
根据以下美食记录，推荐南山附近人均 50 元以下的餐厅：

[粘贴 南山/弹子石、上新街、涂山、南山_豆包.csv 的内容]
```

或者直接把整个目录喂给支持文件上传的 AI。

### 4. 查看表格数据

CSV 文件可以用 Excel 打开（已设置 utf-8-sig 编码，中文不会乱码）。

推荐字段：
- `店名`: 餐厅名称
- `类型`: 火锅/江湖菜/面馆等
- `位置`: 具体地址或区域
- `人均 (元)`: 价格
- `综合评分`: 1-5 分
- `推荐菜品`: 必点菜
- `主要优点` / `主要缺点`: 优缺点分析

## ✏️ 添加新记录

### 🤖 AI 助手必读

**在添加任何内容前，请务必阅读：**
- [AI添加内容指南.md](./AI添加内容指南.md)

**核心原则：**

4. ✅ **验证提交**（Git commit）

---

### 👤 人类用户

### 方法 1：直接编辑

打开任意 `.md` 或 `.csv` 文件，直接添加内容。

### 方法 2：从 Word 转换

```bash
pandoc -f docx -t markdown --wrap=none 新文件.docx > 新文件.md
```

### 方法 3：从 Excel 转换

```python
import pandas as pd
df = pd.read_excel("新文件.xlsx")
df.to_csv("新文件.csv", index=False, encoding="utf-8-sig")
```

### 提交到 Git

```bash
git add 重庆-吃/
git commit -m "添加新的美食记录"
git push
```

## ❓ 常见问题

### Q: 为什么有些文件是空的？

A: 原始文件可能就是空的，转换后保持原样。

### Q: 图片在哪里？

A: 
- Word 内嵌图片 → `media/` 子目录
- 独立图片 → 原位置保留

### Q: 如何更新源文件？

A: 
1. 在 WPS 中修改
2. 重新运行 `python convert_cuisine.py`
3. 检查差异后提交 Git

### Q: CSV 在 Excel 中打开乱码？

A: 不应该出现，我们使用了 `utf-8-sig` 编码。如果仍有问题，用记事本打开另存为 ANSI 编码。
